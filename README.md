# Scrappy Suite

Scrappy Suite is a freeware desktop workspace from [Ravenforge Creations Studio](https://ravenforge.info).

<img width="1200" height="800" alt="Screenshot 2026-05-10 141618" src="https://github.com/user-attachments/assets/80aff73c-d82d-40f1-a627-8a4797bdb094" />

It is built around panels, lightweight modules, and a simple launcher so you can combine writing, files, and planning in one window without a lot of ceremony.

## What It Does

- Opens into a blank workspace so you can build your own layout.
- Lets you add as many panels as you need.
- Loads tools into each panel from the top toolbar.
- Remembers panel state, window state, and launcher shortcuts.

## Included Modules

### Calendar

Use Calendar to create and review events inside Scrappy Suite.

### DirT Writer

Use DirT Writer for drafting, editing, opening, and saving documents.

### Fogre

Use Fogre to browse files, preview content, and open supported files into Writer.

### Help

Use the Help module for an in-app quick start and overview of the workspace.

## Quick Start

1. Launch Scrappy Suite.
2. Click inside a panel to select it.
3. Use the toolbar to load a surface into that panel.
4. Click `Add Panel` to open another empty panel.
5. Open the launcher to save shortcuts for apps, files, folders, or web links.

## Persistence

Scrappy Suite stores several things locally:

- panel layout and loaded modules
- blank panel state
- launcher shortcuts
- window size and position

Documents in DirT Writer should still be saved intentionally when you want a file on disk.
