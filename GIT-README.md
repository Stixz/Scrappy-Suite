# Scrappy Suite
A modular creative framework forged for flexibility, clarity, and expressive workflows.

Scrappy Suite is a collection of tools, modules, and utilities designed to work together as a cohesive creative engine.  
It’s built for makers who want structure without rigidity and power without complexity.

## What’s Inside
- Modular components that can be used independently or together
- A consistent design philosophy across all tools
- A foundation for building your own creative systems

## Philosophy
Scrappy Suite embodies the Ravenforge approach:  
modular, readable, adaptable, and built with intention.
