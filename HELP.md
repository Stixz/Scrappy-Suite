# Scrappy Suite Help

Scrappy Suite is a freeware desktop workspace from Ravenforge Creations Studio. It is built around panels, lightweight modules, and a simple launcher so you can mix tools in one window without a lot of ceremony.

## Quick Start

When Scrappy Suite opens, it starts with a blank panel.

- Click `Add Panel` to open another empty panel.
- Click inside a panel to select it.
- Use the toolbar at the top to load a module into the selected panel.
- Drag panel headers to reorder panels.
- Drag the divider between panels to resize them.
- Use the launcher button to open your shortcuts.

Blank panels are intentional. They let you build only the workspace you want instead of being handed a random arrangement like an unsolicited opinion.

## Modules

### Calendar

Use Calendar to create and review events inside Scrappy Suite.

- View your month and navigate dates.
- Add, edit, and review events.
- Use it for quick planning, reminders, and lightweight scheduling.

### DirT Writer

DirT Writer is the writing and document-editing panel.

- Create and edit rich text content.
- Open supported files into the writer.
- Save documents when you are ready.
- Use it as the main drafting space for notes, documents, and general writing.

### Fogre

Fogre is the built-in file explorer.

- Browse folders and files.
- Preview supported file types.
- Open files into DirT Writer when appropriate.
- Use shortcuts for common locations and basic file operations.

## Saving And Persistence

Scrappy Suite remembers several things automatically.

- Open panels and their loaded modules are restored between sessions.
- Blank panels stay blank until you choose a module.
- Launcher shortcuts are saved locally.
- Window size and position are remembered.

Documents in DirT Writer should still be saved intentionally when you want a file on disk. Local persistence is helpful, but it is not a substitute for saving work you care about.

## Launcher

The launcher gives you quick access to saved shortcuts.

- Open the launcher from the toolbar.
- Add shortcuts for apps, files, folders, or web links.
- Edit or delete shortcuts from the context menu.
- Use `Visit Studio` in the launcher to open Ravenforge Creations Studio online.

## Tips

- Select a panel before loading a module into it.
- `Add Panel` always creates an empty panel first.
- If a panel is empty, use the toolbar to choose what belongs there.
- The footer `About` button opens the app welcome/about view.
- The studio link opens `https://ravenforge.info`.

## Known Limitations

Scrappy Suite is intentionally lightweight, and a few edges may still show.

- Some modules are still evolving and may feel more practical than polished.
- Certain workflows depend on local persistence rather than full project-style document management.
- Behavior may vary if windows are closed during unsaved work prompts.
- This is freeware software, which is another way of saying it tries hard and occasionally develops character.
